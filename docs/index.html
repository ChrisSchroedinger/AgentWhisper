AgentWhisper

Push-to-talk dictation for Linux. Hold a key, speak, release — your words are typed into whatever you're writing.

Transcribed entirely on your own computer with Whisper. No cloud, no account, no audio ever leaving your machine.

Download .deb View on GitHub

How it works

One key. No windows to open, no buttons to click, no waiting for a browser tab.

1

Hold F12

While AgentWhisper runs, the key belongs to it alone — no other program will see it, so nothing fires by accident.

2

Speak

A small translucent panel floats near the bottom of your screen with green bars dancing to your voice — you always know when the mic is live.

3

Release

Your words are typed straight into the window you were working in, and land in your clipboard as backup. A notification confirms what was heard.

Your voice never leaves your computer

Speech recognition runs locally via faster-whisper — OpenAI's Whisper model on your own CPU. One guided ~140 MB model download on first run, then it works fully offline. No account, no API key, no telemetry.

What it looks like

A microphone in your system tray that turns red while recording — and that's all the UI there is.

Small tool, done properly

Everything you need for daily dictation, nothing you don't.

Lives in your tray

Pause and resume, switch modes, toggle auto-type — every control is two clicks away in the tray menu.

Two recording modes

Hold-to-talk (hold F12, release to finish) or press-to-toggle (tap to start, tap to stop). Switch anytime.

Auto-type or clipboard-only

Have text typed into the active window for you, or turn auto-type off and paste with Ctrl+V when you're ready.

Starts with your session

Optional start-at-login from the tray menu or agentwhisper autostart on.

Guided first run

The speech model downloads in the background with live progress in the tray; dictations made meanwhile are queued, not lost.

Yours to configure

A commented config.toml: pick the key, the Whisper model (tiny to medium), recording limits and more.

Scriptable CLI

agentwhisper status / toggle / mode / quit — the same controls from your terminal or keybindings.

Honest diagnostics

agentwhisper status tells you exactly what's missing and how to fix it — including the apt command to run.

Install in two minutes

For Debian/Ubuntu with X11. Everything lands in your home directory — easy in, easy out.

Prerequisite: a few small system packages (one apt line, most preinstalled on XFCE) 
sudo apt install python3-gi python3-gi-cairo gir1.2-ayatanaappindicator3-0.1 \
                 xclip xdotool libnotify-bin

Option A — .deb package

System-wide install, managed by apt.

# download from the latest release, then:
sudo apt install ./agentwhisper_*.deb

Option B — from source

Lives in your home directory, no root needed.

git clone https://github.com/ChrisSchroedinger/agentwhisper.git
cd agentwhisper && ./install.sh

Needs Linux with X11 (not Wayland), Python 3.11+, and a microphone. English models in v0.3.
Full details in the installation guide.
